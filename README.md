# Boat_H2CO3

对Android12 Material You完全支持，确保当前API至少为31才会有取色器效果。
原项目Boat_H2O2已经停更，本项目为它的衍生项目。
本项目与原项目基于BoatAPP完成，所有相关修改请遵守GPL-2协议进行开源。
自定义控件部分引用了McinaBox的部分代码并修改

## 分支更改

增加了新JRE版本支持，精简了一些页面
